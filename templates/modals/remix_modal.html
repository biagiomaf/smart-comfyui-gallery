✨ Generate without even opening ComfyUI.
Tweak your parameters right here and hit Queue. Your image or video will appear in ComfyUI's output folder as if by magic. It's a quick-and-dirty tool, rough around the edges, and won't catch every parameter, so you'll need to know your own workflow. But when it works, it's pretty great.
Some parameters like width, height, or frame count may not appear. This is not a bug; it means they are wired to other nodes in the workflow and cannot be safely overridden from here.

⚙️ Server configuration
Set this environment variable (if different from the default) before starting SmartGallery (e.g., in your shell, .env file, or via the Docker -e flag):
COMFYUI_SERVER_URL — The URL of your ComfyUI instance. Defaults to http://127.0.0.1:8188. This must be set if ComfyUI runs on a different host or port. For example, in Docker, set it to http://host.docker.internal:8188 or the container name. The URL field in this form is pre-filled with this value and can be overridden per session. Click ↺ Reset to restore the default value.
