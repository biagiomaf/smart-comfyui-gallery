💡 Remix Workflow Guide

Welcome to Remix! Generate media directly from your gallery.

Extract the hidden workflows inside your media, tweak parameters, and queue new runs bypassing the heavy ComfyUI interface. Once you craft the perfect setup, save it as a Template for one-click generations.

1️⃣ Simple Tweaks

If your workflow is simple, just use the Auto-Form tab. SmartGallery automatically exposes prompts, seeds, and basic parameters. Edit what you need and hit Queue.

2️⃣ Build a Template

In Auto-Form, click the 📌 Pin icon on the fields you use most. Switch to My Panel to reorder them, then click ⭐ Save to Library to create your custom interface.

3️⃣ Pro Control (Nodepad)

For complex setups (like Ideogram nested JSONs), open the { } Nodepad. Select the core nodes and click ⭐ Favorite. They will appear in 'My Panel' as quick-edit buttons!

Understanding the 3 Views

📝 Auto-Form

The engine room. Intentionally raw, it exposes editable parameters automatically. Find what you need and 📌 Pin it.

🛠️ My Panel

Your dashboard! Populated exclusively with fields you pinned and nodes you favorited. Freely arrange them using ↕ Reorder.

{ } Nodepad

A raw JSON editor for power users. If a custom node isn't supported, edit its raw JSON directly here. Mark nodes as Favorite to access them quickly.

Server Configuration

By default, Remix sends generation jobs to http://127.0.0.1:8188. You can temporarily override this URL using the input field at the top of the window.
